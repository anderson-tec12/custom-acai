# Adicionar copo 200ml ao catálogo

## Contexto

O cardápio é estático em [`src/menu.json`](src/menu.json). Tamanhos atuais: 300ml (R$11), 500ml (R$16), casquinha (R$8). Preço, UI e WhatsApp leem `menu.sizes` via [`src/lib/menu.ts`](src/lib/menu.ts); `SizeId` em [`src/types.ts`](src/types.ts) deriva dos ids do JSON.

## Alterações

### 1. Catálogo — [`src/menu.json`](src/menu.json)

Inserir o 200ml no início de `sizes` (menor → maior → casquinha):

```json
{ "id": "200", "name": "200ml", "price": 9 }
```

Mesmas regras dos copos 300/500: sem `disabledCategories` (frutas liberadas).

### 2. UI — [`src/components/BowlBuilder.tsx`](src/components/BowlBuilder.tsx)

Trocar `grid-cols-3` por `grid-cols-2` na seção de tamanho (~linha 104), para 4 botões em grade 2×2 sem linha desigual.

Default do builder continua `"300"`; sem mudança de lógica em `menu.ts`, carrinho ou WhatsApp.
